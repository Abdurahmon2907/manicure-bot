# Telegram-бот записи для мастера маникюра

Готовый бот на `aiogram` + `SQLite` с:
- записью на 1 месяц вперед через inline-календарь;
- выбором даты/времени, сбором имени и телефона;
- ограничением: у пользователя только 1 активная запись;
- автоосвобождением слота при отмене;
- админ-панелью;
- проверкой подписки на канал перед записью;
- кнопками `Прайсы` и `Портфолио`;
- автонапоминаниями за 24 часа (APScheduler) с восстановлением задач после рестарта.

## Возможности

### Пользователь
- `Записаться`:
  - проверка подписки на канал;
  - выбор даты через inline-календарь;
  - выбор свободного времени;
  - ввод имени и телефона;
  - подтверждение записи.
- `Отменить запись`: отмена своей активной записи.
- `Прайсы`: HTML-сообщение.
- `Портфолио`: inline-кнопка со ссылкой на Pinterest.

### Админ (доступ только по ID)
- `/admin` — открыть админ-панель.
- Добавить рабочий день.
- Добавить/удалить временные слоты.
- Отменить записи клиентов.
- Закрыть день полностью.
- Посмотреть расписание на дату.

## Установка

1. Установите Python 3.11+.
2. В корне проекта установите зависимости:

```bash
pip install -r requirements.txt
```

3. Создайте `.env` в корне проекта (можно скопировать из `.env.example`):

```bash
cp .env.example .env
```

Для Windows PowerShell:

```powershell
Copy-Item .env.example .env
```

4. Заполните `.env`:
- `BOT_TOKEN`
- `ADMIN_IDS` (через запятую, например `123,456`)
- `CHANNEL_ID` (например `-1001234567890`)
- `CHANNEL_LINK` (например `https://t.me/my_channel`)
- `TIMEZONE` (например `Europe/Moscow`)

## Запуск

```bash
python -m manicure_bot.bot
```

## Примечания

- База данных создается автоматически: `manicure_bot/manicure_bot.sqlite3`.
- Напоминание создается только если до визита больше 24 часов.
- При отмене записи задача напоминания удаляется.
- После рестарта бота запланированные напоминания восстанавливаются из БД.
